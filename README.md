📊 Tableau Sales Dashboard | 2026

Project Overview

This project presents an interactive **Sales Dashboard** built using Tableau to analyze key business metrics such as 
Sales, Profit, and Performance Trends. 
The dashboard provides a comprehensive view of business performance across time, categories, and sub-categories,
enabling data-driven insights.
--------------------------------------------------------------------------------------------------------------------------------

 Objectives

Track overall **Sales and Profit performance (2023 vs Previous Year)
Identify top-performing and underperforming product categories
Analyze weekly sales and profit trends
Highlight key business insights for decision-making
